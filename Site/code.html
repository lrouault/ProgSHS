Modélisation du comportement d'un front dans un matériau fibreux

Dans cette partie nous allons expliciter les règles de calcul utilisées lors de la programation de notre projet en Fortran 90.

Règles pour chaque fichier

  • Lister tous les modules utilisés dans le fichier
  • Utiliser Implicit none une seule fois au début

Règles pour les modules

  • Écrire une brève description de ce que l'on trouve à l'intérieur
  • Lister les fonctions présentes
  • Nommer mod_xxx.f90
  • Créer un module par type de fonction
  • Séparation visuelle !****! entre les subroutines/fonctions
  • Mettre un espace après le début et avant la fin d'une subroutine/fonction

Déclaration des variables

  • Aérer la déclaration de nos variables
  • Aligner au maximum les « :: » (avec des espaces autour)
  • Dans une ligne, lister que les variables qui ont un rapport commun (ie propriétés du matériau, données de discrétisation ...)
  • Nom des variables en minuscules sauf pour les tableaux (première lettre en majuscule) et exceptions (température T)
  • Déclarer en dernier les variables utilisées pour les boucles (i j k)

Fonction/Boucle/Condition

  • Tout en minuscule
