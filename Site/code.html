Modélisation du comportement d'un front dans un matériau fibreux

.f90 en général :

  • Lister tous les modules importés (use … / use …)
  • En commentaire en haut, description du module +liste des fonctions contenues
  • Implicit none seulement en début de module
  • Liste de variables avec espaces a, b, c, d

Modules :

  • Nommer mod_xxx.f90
  • Un module par type de fonction ...

Déclaration des variable :

  • Aligner au maximum les « :: » (avec des espaces autour)
  • Dans une ligne, lister que les variables qui ont un rapport commun (ie propriété du matériau / données de discrétisation ...)
  • Nom des variables en minuscules sauf pour les tableaux (première lettre en majusctule) et exeption (température T)
  • integer, dim :: ab
  • Variable i j k .. déclarer à la fin

Fonction/Boucle/Condition :

  • Tout en minuscule (fonction aussi??)
  • Séparation !****! entre les sub/fct
  • Espace après le début et avant la fin d'une subroutine
